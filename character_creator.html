Savage Worlds Web Tools - Character Creator

This web app is a rudimentary Character Creator for the Savage Worlds pen and paper role playing game.

Under Reconstruction

As of August 9th, 2014 this whole section is undergoing a bottom to top rewrite. I'm able to salvage a few things, such as data entry for hindrances and edges, but I'm working on more maintainable code as I've learned a few more tricks in Objest Oriented JavaScript (finally a way to handle inheritance, coming from C++ coding this is a godsend!) since the original creation of this section more than a year ago.

Thanks for your patience
